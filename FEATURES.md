# AOD Suite — Feature Matrix

What's implemented vs planned.

| Feature | Status | Notes |
|---------|--------|-------|
| Toggle AOD on/off | ✅ Working | Via Shizuku settings API |
| Adjust min brightness | ✅ Working | Range 0-255, persists |
| Blurred wallpaper background | ✅ Partial | Works on some devices, issues on Samsung One UI |
| Schedule enable/disable | 🔄 Planned | Time-based auto-toggle |
| Custom clock face | 🔄 Planned | Per-manufacturer support needed |
| Notification badges on AOD | 🔄 Planned | Requires per-app integration |
| Biometric auth on AOD | 🔄 Planned | Face/fingerprint unlock from AOD |
| Battery percentage display | ✅ Working | Via system settings |
| Always-on gesture control | 🔄 Planned | Double-tap to unlock |
| Multi-profile AOD | 🔄 Planned | Different configs per user/app mode |

**Legend:** ✅ = Done · 🔄 = In progress · 📋 = Planned · ❌ = Not feasible
